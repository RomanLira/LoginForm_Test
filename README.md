# LoginForm Test Project

This is a single-page ASP.NET Core MVC application with a login form that authenticates users via a SOAP web service.

This is a test application for mekashron.com.

## Features

- Login form with **Email** and **Password** fields.
- SOAP service integration:
  - Calls `Login` method of the external web service.
  - Displays success or error messages.
  - Shows entity details on successful login.
- Fully responsive layout using **Bootstrap 5**.
- Client-side validation with **ASP.NET Core validation**.
- Dockerized for easy deployment on cloud platforms like **Render**.
